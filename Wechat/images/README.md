# TabBar图标目录

请将以下图标文件放置到此目录：

- tab_home.png / tab_home_active.png (首页图标)
- tab_book.png / tab_book_active.png (词书图标)  
- tab_community.png / tab_community_active.png (社区图标)
- tab_profile.png / tab_profile_active.png (我的图标)

可使用 https://www.iconfont.cn 下载适合的图标。

如果暂时没有图标，可以将app.json中的iconPath和selectedIconPath删除或注释掉以使用文字模式。
